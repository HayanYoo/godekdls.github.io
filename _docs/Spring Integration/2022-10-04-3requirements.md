## 1.1. Compatible Java Versions

Spring Integration 5.5.x에서 호환되는 자바의 최소 버전은 Java SE 8이다. 이전 자바 버전은 지원하지 않는다.

---

## 1.2. Compatible Versions of the Spring Framework

Spring Integration 5.5.x는 스프링 프레임워크 5.3 이상이 필요하다.
