Spring Cloud Stream은 아래와 같은 메세징 미들웨어, 스트리밍 플랫폼을 위한 바인더를 제공한다:

- RabbitMQ
- Apache Kafka
- Amazon Kinesis
- Google Cloud PubSub
- Solace's PubSub+
- Azure Event Hubs
